Data Science Engineer 직무에 지원했던 곳에서 연락이 왔다. 그리고 Homeworking 문제를 받았다. 하지만, 전혀 나는 손을 될 수가 없었다. 지난 시간동안 나는 이론적 지식도 많이 잊었고 데이터는 거의 손을 되지 않았다.

결정해야 한다. 어떤 직무를 이직을 할지. 현실적으로 Data Scienctist 직무는 어려워 보인다. 아니 현실 가능성이 거의 제로에 가깝다. 그나마 가장 익숙한 직무는 Data Engineer 직무다. 분명 몇 주 전에 깨달았는데도 허영의 마음과 욕심에 잊고 있었다. 하지만 Machine Learning은 공부하자.

도메인, Automotive 분야로 이직하면 가장 이상적이겠지만, 그렇다고 내가 자동차 지식을 연구소 사람들만큼 해박하게 가지고 있지도 않다. 더군다다 긴 시간동안 자동차 데이터는 만져보지도 못 했다. 상관없다. 어떤 도메인이든. 냉정해지자. 독일로 가는 것이 목적이고 목표는 영주권과 창업이다. 냉정해지자.

정리하면,

* Data Scientist
  * 통계/수학
  * Machine Learning
  * 분석 경험, 분석 경험이 없다면 Kaggle 경험
  * Data Engineer

* 데이터 플랫폼 지식: Spark, Kafka
  * 전처리 경험: Scala, Python
  * Production 운영경험
  * Coding Test

따라서, 내가 학습할 것은 정해졌다. 이제 한 눈 팔지말고 제대로 하자.
